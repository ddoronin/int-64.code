# Create draft invoice (minimal)

Generated by Postman Code from PayPal's public Postman collection.

## Endpoint (Sandbox)

`POST https://api-m.sandbox.paypal.com/v2/invoicing/invoices`

## Auth

Bearer token: `Authorization: Bearer <access_token>`

## Minimal payload

```json
{
  "detail": {
    "invoice_date": "2026-02-16",
    "currency_code": "USD"
  }
}
```

## Usage

```js
import { createDraftInvoiceMinimal } from './client.js';

const invoice = await createDraftInvoiceMinimal({
  baseUrl: 'https://api-m.sandbox.paypal.com',
  accessToken: process.env.PAYPAL_ACCESS_TOKEN,
  input: { invoiceDate: '2026-02-16', currencyCode: 'USD' },
});

console.log(invoice.id, invoice.status);
```
